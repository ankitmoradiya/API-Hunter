We request that you **DO NOT** open a public GitHub issue for security vulnerabilities. Public disclosure before a fix is available can put users at risk.

Please report security vulnerabilities privately to the maintainer via email:

*   **Email:** `[Your Preferred Security Email Address]` (e.g., `security@ankitmoradiya.com` or a personal email)
*   **Subject:** `[SECURITY] API-Hunter Vulnerability Report`

### What to Include in the Report

To help us quickly understand and resolve the issue, please include the following details:

1.  **A clear description** of the vulnerability.
2.  **Steps to reproduce** the vulnerability (including any necessary configuration or command-line arguments).
3.  **The version** of API-Hunter affected.
4.  **Your name/handle** (if you wish to be credited).
